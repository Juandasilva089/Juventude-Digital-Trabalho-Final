Quartos

Tipos

  • Quarto duplo ou twin – quarto para duas pessoas. O elemento característico é que este quarto tem duas camas individuais.

  • Quarto de casal – é também um quarto para duas pessoas, mas com uma cama de casal de tamanho “king”.

  • Quarto Triplo – quarto para 3 pessoas, normalmente com 3 camas individuais.
